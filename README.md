# 🖥️ Portfolio — Luiz Reis Neves

A personal portfolio built with a dark/matrix aesthetic, featuring interactive animations, a typewriter terminal effect, and a synchronized skills carousel.

## 🔗 Live Demo

[stellar-meerkat-01aa60.netlify.app](https://stellar-meerkat-01aa60.netlify.app/)

## ✨ Features

- Matrix rain effect with canvas
- IBM-style terminal animation for skills
- Synchronized skills carousel
- 3D tilt effect on the hero terminal
- Scroll-triggered animations with GSAP
- Contact form with EmailJS integration
- Fully responsive layout

## 🛠️ Tech Stack

- React
- TypeScript
- Tailwind CSS
- GSAP + ScrollTrigger
- EmailJS
- Vite
- Devicons React
- Lucide React

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/Luiz-Reis-Neves/portfolioFc.git

# Navigate to the project folder
cd portfolioFc

# Install dependencies
npm install

# Start the development server
npm run dev
```
